# OpenRA SHOCKWAVE — SW_ORA 0.001 (release 2026.07.04)

**Playable Windows build** of OpenRA (Red Alert mod with SHOCKWAVE changes).

This repo is the **game**, not the source code. Download, extract, and run — no compiling.

## Quick start (Windows)

1. Download this repo as a ZIP (**Code → Download ZIP**) or clone it.
2. Extract to a folder (e.g. `C:\Games\OpenRA-SHOCKWAVE`).
3. Double-click **`launch-game.cmd`** and type **`ra`** for Red Alert.

Or run directly:

```cmd
launch-game.cmd Game.Mod=ra
```

## Requirements

- **Windows 10/11 (64-bit)**
- **[.NET 8 Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/8.0)** (Runtime, not SDK — one-time install if you do not already have it)

On first launch, OpenRA may ask you to install **Red Alert game content** (original game files or the free content download). That is normal.

## What is included

- `bin\OpenRA.exe` — compiled game engine
- `mods\` — Red Alert and related mod data
- `launch-game.cmd` — start the game
